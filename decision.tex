\section{Back end}\label{sec:backend}
\begin{description}
\item[Java as programming language]
Reason: it has APIs for MongoDB, Kafka, and websockets and is well supported by development tools (testing, deployment etc.).
\item[feeding the data from Kafka to the MongoDB and having the Hub access only the MongoDB]
Reason: to simplify data processing (filtering, aggregating).
\item[Apache Tomcat as webserver]
Reason: established platform for web applications, integrates well into a java-oriented development environment.
\end{description}
